# בחינת אמצע במימון

אפליקציה סטטית לבחינת אמצע במימון. כל סטודנט מזין תעודת זהות, מקבל גרסה דטרמיניסטית שונה של הבחינה, והציונים נשמרים ב-Google Sheet דרך Google Apps Script. בבחינה 11 שאלות, כל שאלה 10 נקודות, ולכן הציון המקסימלי הוא 110.

## מה כלול

- `index.html` - דף הבחינה לסטודנטים.
- `app.js` - טעינת שאלות, טיימר, איסוף תשובות והגשה.
- `styles.css` - עיצוב רספונסיבי בעברית.
- `config.js` - מקום להדבקת קישור ה-Apps Script.
- `backend/Code.gs` - קוד השרת שמייצר שאלות, בודק תשובות ושומר ציונים.

## פרסום מהיר

1. פתח Google Sheet חדש.
2. בתפריט Extensions בחר Apps Script.
3. הדבק את כל תוכן `backend/Code.gs`.
4. שנה את `CONFIG.secret` לטקסט פרטי ואקראי.
5. Deploy -> New deployment -> Web app.
6. בחר Execute as: Me, וגם Who has access: Anyone with the link.
7. העתק את כתובת ה-Web app.
8. הדבק אותה ב-`config.js` בתוך `appsScriptUrl`.
9. העלה את התיקייה ל-GitHub והפעל GitHub Pages.

## חלון הבחינה

החלון מוגדר בקובץ `backend/Code.gs`:

```js
openAt: "2026-06-15T09:50:00+03:00",
closeAt: "2026-06-15T10:30:00+03:00"
```

## ציונים

הגיליון יקבל אוטומטית שורות עם:

- זמן הגשה
- תעודת זהות
- ציון
- ציון מקסימלי
- מספר אישור
- פירוט בדיקה
- התשובות שהוזנו

## הערה חשובה

הבדיקה נעשית ב-Apps Script ולא בדפדפן, כדי שהתשובות לא יהיו בקוד האתר שחשוף לסטודנטים. עדיין מומלץ לא לפרסם את קוד ה-Apps Script עצמו לסטודנטים.
